# Mi Proyecto de Prueba - App de Productos

[cite_start]Esta es la app que hice para la prueba técnica usando Expo y React Native[cite: 4, 8]. [cite_start]La aplicación se conecta a una API para mostrar una lista de productos y permite ver la información de cada uno al hacer clic[cite: 9].

## Lo que hace la app
* [cite_start]**Lista de inicio**: Trae los productos de la API de FakeStore[cite: 12, 15].
* [cite_start]**Detalles**: Cuando tocas un producto, te lleva a otra pantalla con su foto, precio y descripción[cite: 18, 19].
* [cite_start]**Navegación**: Use Stack Navigator para moverme entre pantallas.
* [cite_start]**Carga**: Puse un circulito de carga (ActivityIndicator) para cuando los datos están bajando[cite: 27].
